# Peel-Testing-Machine-Capstone-Project
A low-cost peel testing machine developed to measure adhesive bond strength for industrial and research applications.  
This project focused on mechanical system design, prototyping, fabrication, and testing, with the goal of creating an affordable alternative to expensive commercial testing systems.

## 🚀 Project Highlights
- Developed as a final-year capstone project
- Patent filing initiated for the innovation
- Winner at GPND Project Exhibition
- Designed and fabricated complete mechanical structure
- Built for real-world adhesive strength testing applications

## 🛠️ Tools & Technologies
- SolidWorks
- Fusion 360
- Mechanical Fabrication
- Prototyping & Testing
- Engineering Design

## ⚙️ Key Features
- Low-cost and compact testing setup
- Accurate adhesive peel strength evaluation
- End-to-end mechanical system development
- Structural design optimized for stability and repeatability
- Prototype validated through physical testing

## 📌 Engineering Contributions
